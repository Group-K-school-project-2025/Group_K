About Our Project

This website project was created as part of a course at Oulu University of Applied Sciences (OAMK). The goal of the project was to design and build a modern, responsive web interface using HTML, CSS, Bootstrap and other front-end technologies.

It showcases various pages including a home page, contact form, login and registration system, all with a consistent theme and design. The project helped us learn how to implement design principles, work with responsive layouts, and deliver a user-friendly experience.
